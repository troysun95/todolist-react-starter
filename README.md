# A8:Todo List

## 介紹

運用 React 架設的 to-do-list，使用者可瀏覽，或更新待辦事項數目或內容。

## 功能

- 瀏覽所有待辦事項
- 新增 / 刪除 待辦事項
- 修改 待辦事項內容
- 統計 事項總數

## 開始使用

1. 請先確認有安裝 node.js 與 npm
2. clone 專案至本地資料夾
3. 本地終端機開啟至專案儲存的資料夾

   ```jsx
   npm install
   ```

4. 安裝完畢 繼續輸入

   ```

   npm run start
   ```

5.於瀏覽器畫面輸入以下網址，前往 待辦事項頁面。
http://localhost:3000/todo

6. 若欲停止伺服器，於終端機輸入

   ```jsx
   ctrl + c;
   ```

## 開發工具

- React @ 18.2.0
- clsx @ 1.2.1
- styled-components @ 5.3.5
